夜间模式登场。

在很暗的光线下拍照，也能光彩照人。全新推出的夜间模式会在弱光环境下自动打开，让夜拍照片更亮眼。不需要开闪光灯，拍出的颜色照样自然，画面照样亮丽。

绚丽夜拍背后的力量。

如何在低光环境下拍出精彩大片，一直都是个摄影难题。这是因为快门需要开启更长的时间，但画面却可能会因此晃动模糊。既要保留阴影部分的细节，又要避免亮部曝光过度，也很棘手。就更别提还要保证颜色的自然了。

在 iPhone 11 和 iPhone 11 Pro 上，新的广角摄像头传感器能够配合智能软件和 A13 仿生，即使在很暗的光线下，也能拍出具有丰富细节的绚丽照片，实现 iPhone 前所未有的拍摄效果。

夜间模式会在需要时自动开启。比如当你在烛光晚餐时想要拍照片，这时轻点快门按钮，摄像头会拍摄多张照片，同时光学图像防抖功能会保持镜头稳定。

接下来，轮到相机软件上场。它会协调多张图像，修正画面晃动；舍掉过于模糊的部分，使用比较清晰的部分来拼合；调节对比度，让所有元素保持平衡；精细调整色彩，使它们看起来真实自然；然后智能地去除噪点，增强细节，生成最终的图像。

这一切带给你的，是令人眼前一亮的夜拍照片：更丰富的细节，更少的噪点，更贴近彼时彼刻的现场感。

拍摄 60 fps 的 4K 视频，
这种级别的运算能力可不简单。

我们打造 A13 仿生，是为了带来更快的速度和先进的机器学习技术。也正是它，让 iPhone 11 拥有强大的处理能力，以应对各种需要大量处理器资源的任务。举个例子，以 60fps 的帧率拍摄扩展动态范围的 4K 视频时，就像有无数条信息同时涌入视频编码器一样。处理如此海量的信息，对大多数芯片来说都是一个挑战，但 A13 仿生做起来游刃有余。

我们可以将视频看成一连串的画面。在速度惊人的摄像头感光元件的帮助下，iPhone 11 能以每秒 120 帧的速度，交替生成标准曝光和短曝光画面。

图像信号处理器和视频编码器会瞬间分析每帧画面，捕捉尽可能多的细节。为了实现更好的效果，神经网络引擎会运用实时机器学习技术来优化场景的不同部分。例如，它会重新调整前景人物的亮度，同时减少噪点，增强天空的色彩。一切，都是在瞬间自动完成。

要实现 iPhone 最高的视频画质绝非易事。但超强的 A13 仿生，会让你大开眼界。

在面容 ID 眼里，你不过是数字而已。这绝对是件好事。

每一件 Apple 产品从设计开始，就以保护你的隐私为己任。正因如此，面容 ID 不会存储或分享你的照片。取而代之的，是利用先进技术，准确地绘制你的面部几何结构图，以此进行身份确认。

下面来看看它的运作方式。首次设置面容 ID 时，原深感摄像头系统会将 3 万多个肉眼不可见的光点投射在你面部，并通过红外镜头读取点阵图案。A13 仿生随后会为你绘制出独一无二的深度图，并将图中的角度和线条转换成数学表达。这一切为你打造出自己的那把“万能钥匙”，并在你设备上的安全隔区中加密保护。它只会保留在 iPhone 上，不会备份到 iCloud 或 Apple 的服务器上。

当你解锁手机 (或登录 app、使用 Apple Pay) 时，面容 ID 会执行相同的流程，投射光点，并与你存储的数据进行比较。如果两者一致，就会确认你的身份。面容 ID 甚至能适应你的样貌变化。它还具有注视感知功能，因此只有当你直视屏幕时，它才会为你解锁 iPhone。

种种保障，让面容 ID 成为安全性出众的身份验证技术。现在，面容 ID 还变得更快，更好用，能以更多角度和更大距离来进行识别。
